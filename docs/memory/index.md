## 書き方の指針

- **動くコードを残す**: 「こうすれば動いた」を必ず実例つきで
- **バージョン情報**: Blenderのバージョンに依存する内容は明記
- **失敗例も書く**: 「これは動かなかった」も価値ある情報
- **リンクをつける**: 参考にした記事・公式ドキュメントのURL
