## 構成パーツ

| パーツ | 形 | 技法 |
|---|---|---|
| 座面 | 角丸ボックス | Cube + Bevel |
| 脚 ×4 | 円柱 | Cylinder + ループ配置 |
| 背もたれ | 角丸ボックス | Cube + Bevel |
| 床 | Plane | プリミティブそのまま |

## 設計のポイント

### パラメータ駆動

寸法をすべて定数（`SEAT_W`, `LEG_R`, `BACK_H`...）に切り出してある。後で「もっと脚を太くしたい」「背もたれを高くしたい」と思ったときに、その値だけ書き換えればいい。

### 4本の脚を for ループで

`for sx, sy in [(1,1), (1,-1), (-1,1), (-1,-1)]:` の符号反転で4隅を表現。**Mirror モディファイアと同じ発想だが、独立したオブジェクトで4本作りたいときはループのほうが扱いやすい**。

### add_rounded_box ヘルパー関数

座面と背もたれは「位置とスケールが違うだけの角丸ボックス」なので、関数化して2回呼んでいる。**繰り返しパターンを見つけたら関数化する** のは Phase 2 以降のキーポイント。

## 改造アイデア

- 脚を **テーパー（先細り）** にしたい → Cylinder の代わりに Cone を使う
- 背もたれに **横棒（スポーク）** を入れる → Array モディファイアで Cube を等間隔複製
- **ダイニングセット** にする → 椅子を for ループでテーブルの周りに配置（Phase 1 のグリッド配置応用）
- **マテリアル** を付けたい → Phase 3（マテリアル編）の課題に
