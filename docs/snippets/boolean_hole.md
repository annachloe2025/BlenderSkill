## ブーリアンの3モード

`m.operation` には3種類:

| モード | 意味 | 例 |
|---|---|---|
| `'DIFFERENCE'` | A から B を引く（穴あけ）| 立方体に丸穴 |
| `'UNION'` | A と B を合体 | 2つの形を融合した不規則な塊 |
| `'INTERSECT'` | A と B の重なる部分だけ残す | 球と立方体の交差で角丸の塊 |

## 使い方の流れ

1. **ベース** と **カッター（切り取る側）** の2オブジェクトを用意
2. ベースに `BOOLEAN` モディファイアを追加し、`object` 引数にカッターを指定
3. `bpy.ops.object.modifier_apply()` で実メッシュ化
4. カッターは不要になるので `bpy.data.objects.remove()` で削除

## ソルバの選択

`m.solver` には `'FAST'` と `'EXACT'`:

- `FAST`: 速いが同じ位置に頂点があると壊れやすい
- `EXACT`: 計算が重いが綺麗。**学習中は EXACT 推奨**

## 落とし穴

- **カッターはベースより少し大きく** すること。ぴったり同じだと境界が壊れる（`depth=4` で Cube の幅 2 に対して長めにしてある）
- ブーリアン後のメッシュは N-Gon（多角形面）が増える。スムーズシェードが崩れることがあるので、**仕上げにベベルで軽く整える** とよい
